# C - Simple Shell

This project is about making our own simple shell command language interpreter which emulates the basic functionality.

## Compilation

```bash
$ gcc -g -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o hsh
$ ./hsh
```
